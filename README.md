# CodeMate Project

## 1. Collaborative Code Editor

### Overview
CodeMate provides a real-time collaborative coding environment where multiple users can write, edit, and execute code together seamlessly.

### Key Features

- **Real-Time Collaboration**  
  Multiple users can edit the same code simultaneously with instant updates using **Django Channels** and **WebSockets**.

- **Secure Code Execution**  
  Each user’s code runs inside a separate **Docker container**, ensuring isolated and secure execution environments.

- **Customizable Environments**  
  Users can install project-specific libraries and packages, allowing flexibility for different programming tasks and technologies.

- **Efficient Syncing**  
  Optimized real-time synchronization minimizes latency and reduces editing conflicts for a smooth collaborative experience.

## 2. Coding Lab Test System

### Overview
The Coding Lab Test System is an intelligent, teacher-monitored coding assessment platform integrated into CodeMate. Teachers can create timed coding tests, monitor student activities in real time, and evaluate submissions with AI-assisted analytics.

### Key Features

- **Teacher Dashboard**  
  Teachers can create coding questions with configurable time limits and monitor enrolled students during live assessments.

- **Student Monitoring**  
  Each student’s coding activity is tracked separately. Any pasted content is recorded and visible only to the teacher for maintaining academic integrity.

- **AI-Powered Evaluation**  
  Using **LangChain**, teachers can analyze submitted code for:

  - Time complexity  
  - AI-generated code detection  
  - Correctness and validity  

- **Post-Test Analysis**  
  Teachers can review:

  - Typing speed  
  - Code quality  
  - Submission history  
  - Manual grading and feedback  

### Impact
The Coding Lab Test System enhances academic integrity, provides actionable insights for teachers, and creates a monitored, interactive coding assessment environment for students.
